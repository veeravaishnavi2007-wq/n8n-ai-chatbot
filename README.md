# n8n-ai-chatbot
# 🤖 AI Chatbot using n8n

## 📌 Project Overview

This project is an AI-powered chatbot built using **n8n workflow automation**.  
The chatbot uses an AI language model to understand user queries and generate intelligent responses.

The workflow connects user input with an AI Agent and LLM model to provide automated conversations.

---

## 🚀 Features

- AI-based conversation
- Automated response generation
- Workflow automation using n8n
- LLM integration
- Webhook-based communication
- Real-time chatbot responses

---

## 🛠️ Technologies Used

- n8n (Workflow Automation)
- AI Agent
- Google Gemini API
- Webhooks
- Large Language Model (LLM)

---## ⚙️ Workflow
User Input

|

|

Response Output

## 🔧 How to Use

1. Install or open n8n
2. Import the `chatbot-workflow.json` file
3. Add your own API key
4. Configure credentials
5. Activate the workflow
6. Start chatting with the AI bot
